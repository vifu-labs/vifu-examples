# External AI Games

These examples show how to bring existing AI-native games to Vifu with minimal changes.

## Examples

- [AIventure](aiventure): Angular + Phaser AI adventure game adapted through a Vifu manifest and model-backend adapter.

AIventure is included as a Git submodule so its upstream history remains inspectable:

https://github.com/vifu-labs/AIventure

If the folder is empty after cloning:

```bash
git submodule update --init --recursive
```
